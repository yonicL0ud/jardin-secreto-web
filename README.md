Jardín Secreto - Tienda de Plantas

Descripción del negocio:
Jardín Secreto es una tienda virtual de plantas de interior, suculentas y accesorios de jardinería

Público objetivo: Personas que buscan decorar sus hogares con plantas, desde principiantes hasta jardineros experimentados.
